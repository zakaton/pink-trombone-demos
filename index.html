Pink Trombone Demos

A collection of demos using Neil Thapen's Pink Trombone.

The webpages work by communicating with each other using the WebSockets API or the BroadcastChannel API, so some pages need other pages open to work, e.g. the Text-to-Speech webpage sends messages to the Pink Trombone webpage.

Demos

Pink Trombone

Opens the "Pink Trombone" Speech Synthesizer. Receives messages from the other webpages to control it. Also includes a dark mode and multi-channel mode (you won't need to use these versions)

Debug

Used for sending test messages to Pink Trombone

requires Pink Trombone

Pitch

Gets the microphone's Pitch and Volume and sends it to Pink Trombone.

requires Pink Trombone

Machine Learning (Meyda.js MFCC) (Essentia.js Mel Spectrogram)

Uses ML5.js and Meyda.js's MFCC (or Essentia.js's Mel Spectrogram) to create a neural network that maps speech to Pink Trombone parameters.

requires Pink Trombone

KNN Classifier (Meyda.js MFCC) (Essentia.js Mel Spectrogram)

YouTube Video

Uses ML5.js and Meyda.js's MFCC (or Essentia.js's Mel Spectrogram) to create a KNN Classifier that maps speech to Pink Trombone parameters.

requires Pink Trombone

3D Navigation

YouTube Video

Moves around an A-FRAME scene using the KNN Classifier, using classes forward, backward, left, and right

requires Pink Trombone and KNN Classifier

Text to Speech

YouTube Video

Converts text to timed phonemes, and then sends the timed phonemes to the Pink Trombone.

requires Pink Trombone

Avatar Lip Sync

YouTube Video

Receives timed phonemes from Text to Speech and animates a Ready Player Me avatar's mouth.

requires Pink Trombone and Text to Speech

Pronunciation

YouTube Video

Displays the pronunciation of a word, and speaks it aloud as one mouses over the phonemes. One can also speak the phonemes aloud to "read" the word

requires Pink Trombone and KNN Classifier

Rhyming Dictionary

YouTube Video

Type in a phrase and see all the possible rhyming phrases

requires Pink Trombone

Robot.js Controller

YouTube Video

Uses Robot.js to control your computer's mouse/keyboard. Requires the KNN Classifier, using classes up, down, left, right, mousedown, and mouseup

Try on tldraw (for drawing) or bruno simon car demo (for arrow keys)

requires WebSocket version and KNN Classifier
